# 📚 Data Structures and Algorithms in C++ (DSA-1)

A day-wise collection of Data Structures and Algorithms implemented in C++, covering core topics like arrays, linked lists, stacks, queues, sorting, trees, and more.

---

## 📅 Day-wise Progress

| Day | Topics Covered |
|-----|----------------|
| **Day 01** | Introduction to C++ for DSA — Variables, Data Types, Input/Output, Basic Operators |
| **Day 02** | Arrays — Declaration, Initialization, Traversal, Insertion, Deletion |
| **Day 03** | Arrays — Searching (Linear Search, Binary Search), 2D Arrays |
| **Day 04** | Arrays — Sorting (Bubble Sort, Selection Sort, Insertion Sort) |
| **Day 05** | Strings — Character Arrays, String Operations, String Functions |
| **Day 06** | Functions & Recursion — Recursive Problems, Factorial, Fibonacci |
| **Day 07** | Pointers — Pointer Basics, Pointer Arithmetic, Dynamic Memory Allocation |
| **Day 08** | Structures & Classes — OOP Basics, Class Definition, Constructors |
| **Day 09** | Linked List — Singly Linked List — Insertion, Deletion, Traversal |
| **Day 10** | Linked List — Doubly Linked List & Circular Linked List |
| **Day 11** | Stack — Array-based Stack, Stack Operations (Push, Pop, Peek) |
| **Day 12** | Stack — Linked List-based Stack, Applications (Parenthesis Matching, Infix/Postfix) |
| **Day 13** | Queue — Array-based Queue, Circular Queue, Queue Operations |
| **Day 14** | Queue — Linked List-based Queue, Priority Queue, Deque |
| **Day 15** | Sorting Algorithms — Merge Sort, Quick Sort, Analysis of Sorting |
| **Day 16** | Introduction to Trees — Binary Tree, Tree Traversals (Inorder, Preorder, Postorder) |

---

## 🏗️ Mini Project

A small project applying one or more of the core data structures (e.g., a menu-driven application using Linked List, Stack, or Queue) to demonstrate real-world usage of concepts learned throughout the course.
